# Campus Ride - GEU Bus Management System

A smart bus management system built for **Graphic Era Deemed to be University, Dehradun**. Campus Ride streamlines campus transportation by connecting students, drivers, and administrators on a single platform.

## Features

- **Student Portal** – Book seats, track buses in real-time, and view booking history
- **Driver Dashboard** – View assigned routes, schedules, and manage trips
- **Admin Panel** – Manage buses, routes, drivers, students, and schedules
- **Live Map** – OpenStreetMap integration centered on GEU campus
- **Role-Based Access** – Separate dashboards for students, drivers, and admins

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- Framer Motion
- Lovable Cloud (backend)

## Getting Started

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd campus-ride

# Install dependencies
npm i

# Start the development server
npm run dev
```
