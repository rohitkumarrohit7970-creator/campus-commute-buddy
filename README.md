# Campus Ride 🚌

**Campus Ride** is a comprehensive bus transportation management system designed exclusively for **Graphic Era Deemed to be University (GEU), Dehradun**.

## About Campus Ride

Campus Ride simplifies and digitizes the entire campus bus transportation experience. Whether you're a student trying to catch the morning bus, a driver managing your route, or an administrator overseeing the fleet — Campus Ride has you covered.

The platform provides real-time bus tracking, seat booking, route management, and schedule coordination — all in one place.

## Who Is It For?

### 🎓 Students
- Browse available bus routes and timings
- Book seats on preferred buses in advance
- Track buses live on the map
- View and manage booking history

### 🚐 Drivers
- View assigned routes and stops
- Check daily and weekly schedules
- Monitor passenger count and seat availability

### 🛠️ Administrators
- Add, edit, and manage the entire bus fleet
- Create and modify routes with multiple stops
- Assign drivers to buses and routes
- Manage student records and bookings
- View real-time dashboard with fleet statistics

## Key Highlights

- **Real-Time Bus Tracking** — Live map powered by OpenStreetMap, centered on the GEU Clement Town campus
- **Smart Seat Booking** — Students can reserve seats ahead of time to guarantee a spot
- **Role-Based Dashboards** — Dedicated interfaces for students, drivers, and admins
- **Route Timeline** — Visual route display showing all stops in order
- **Fleet Overview** — Admin dashboard with active buses, total routes, and occupancy stats

## Tech Stack

| Technology | Purpose |
|---|---|
| React + TypeScript | Frontend framework |
| Vite | Build tool & dev server |
| Tailwind CSS | Utility-first styling |
| shadcn/ui | UI component library |
| Framer Motion | Animations & transitions |
| Recharts | Dashboard charts & analytics |
| OpenStreetMap | Live map integration |
| Lovable Cloud | Backend, database & auth |

## Getting Started

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate into the project
cd campus-ride

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at `http://localhost:8080`.
